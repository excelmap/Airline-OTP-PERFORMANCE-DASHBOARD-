# Airline-OTP-PERFORMANCE-DASHBOARD
# ✈️ Airline On-Time Performance (OTP) Dashboard | Power BI

## Project Overview

This project presents an interactive **Airline On-Time Performance (OTP) Dashboard** built in **Power BI** to analyze airline operational efficiency across Australia. The dashboard enables users to monitor key performance indicators, identify operational trends, compare airline performance, and evaluate flight reliability through an intuitive and interactive reporting experience.

Using historical airline operational data, the dashboard transforms millions of flight records into meaningful business insights that support data-driven decision-making for airline management, airport authorities, and operations teams.

---

## Business Objective

The objective of this project is to provide a comprehensive view of airline operational performance by measuring on-time performance, delays, cancellations, and flight activity. The dashboard helps stakeholders:

* Monitor airline operational efficiency.
* Track Departure and Arrival On-Time Performance (OTP).
* Analyze cancellation and delay trends over time.
* Compare airline market share based on total flights.
* Identify the best-performing routes.
* Evaluate monthly flight operations and scheduling performance.
* Support strategic and operational decision-making.
